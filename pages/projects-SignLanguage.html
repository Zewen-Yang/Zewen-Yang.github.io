AI-SignLanguage

AI-SignLanguage — AI-Enabled Sign Language Understanding and Learning — bridges the communication gap. This project aimed at bridging the communication gap between hearing-impaired individuals and the hearing world. The project utilizes machine learning techniques to develop intelligent and personalized human-computer interfaces that enable seamless and natural interactions between people and technology.

The project involves several machine learning techniques, including computer vision and deep learning, to develop an intelligent system that can accurately recognize and interpret sign language gestures. Object detection techniques allow the system to identify and track the various sign languages. Additionally, using action detection techniques enables the system to analyze and interpret the temporal patterns of sign language gestures. Finally, a sign language learning application that utilizes gesture analysis evaluated by human experts is develpoed. This approach enables the system to provide personalized feedback and guidance to users, helping them improve their sign language skills and proficiency levels.

More details about this project can be found in the slides.

Data Scientist

  • Sign language recognition with object detection using TensorFlow2 Detection Models and the state-of-the-art model YOLOv8
  • Sign language understanding with action detection with MediaPipe and LSTM model
  • Develop a sign language learning application for gesture analysis with by human experts' evaluation
